# La Fonda de Doña Licho

Sitio web para restaurante de comida mexicana tradicional.

## Instrucciones para agregar imágenes

Necesitas agregar las siguientes imágenes en la carpeta `images/`:

### Página de Inicio (index.html)
- `logo-small.png` - Logo pequeño de la fonda (casa estilo mexicano)
- `taco.png` - Imagen de tacos
- `enchiladas.png` - Imagen de enchiladas
- `tostadas.png` - Imagen de tostadas
- `quesadilla.png` - Imagen de quesadilla

### Página de Menú (menu.html)
- `enchiladas-verdes.jpg` - Enchiladas verdes
- `tacos-pastor.jpg` - Tacos al pastor
- `sopes.jpg` - Sopes
- `flautas.jpg` - Flautas
- `birria.jpg` - Birria con arroz

### Página Conócenos (conocenos.html)
- `dona-licho.jpg` - Foto de Doña Licho (señora mayor sonriendo, blusa azul)

### Página Galería (galeria.html)
- `jarra-barro.jpg` - Jarra de barro sobre sarape
- `decoracion.jpg` - Decoración mexicana colorida
- `clientes.jpg` - Clientes comiendo
- `restaurante.jpg` - Interior del restaurante
- `chef-preparando.jpg` - Chef preparando comida
- `sombrero-rosa.jpg` - Sombrero rosa en silla con sarape
- `guacamole.jpg` - Plato de comida con guacamole
- `mole-arroz.jpg` - Mole con arroz

### Página Contacto (contacto.html)
- `chef-plato.jpg` - Chef con mandil mostaza sosteniendo un plato

## Estructura del sitio

- `index.html` - Página de inicio
- `menu.html` - Menú de platillos
- `conocenos.html` - Historia de la fonda
- `galeria.html` - Galería de fotos
- `contacto.html` - Información de contacto
- `styles.css` - Estilos CSS
- `images/` - Carpeta de imágenes

## Ver el sitio localmente

Simplemente abre cualquier archivo HTML en tu navegador.
